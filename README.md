# apparel-inventory-order-api

## 概要
アパレルショップ向けの在庫・発注管理 REST API です。

## 使用技術
- Java
- Spring Boot
- Spring Security
- OAuth2.0 / OpenID Connect
- Keycloak
- JWT
- MyBatis
- PostgreSQL
- Flyway
- Spring Batch
- Docker Compose
- AWS ECS / RDS / S3 / CloudWatch / ALB

## 主な機能
- 商品管理
- 店舗別在庫管理
- 入荷登録
- 発注管理
- 日次在庫集計バッチ
- 在庫不足商品抽出
- 発注候補CSV出力
- Keycloak + JWT による認証・認可
